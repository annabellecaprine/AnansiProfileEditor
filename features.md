# Anansi Profile Editor - Features

## Core Capabilities
- **Real-Time Visual Editor**: See changes instantly as you edit your profile theme.
- **One-Click Export**: Generates a single block of optimized CSS + HTML ready to paste directly into JanitorAI's "Bio" field.
- **State Persistence**: Automatically saves your work (theme, content, CSS) locally, so you never lose progress.

## Styling & Theming
- **Global Theme Control**:
  - **Accent Color**: Sets the primary highlight color for borders, badges, and glows.
  - **Text Color**: Overrides default text color for all profile elements.
    - **Backgrounds**: Set custom page background images and card background colors with adjustable opacity and blur.

## Advanced Layout Engine
  - **Flexbox Controls**: Fine-tune the alignment (Row/Col, Start/Center/End) of the Header, Bio, and Stats sections.
  - **Spacing**: precise margin and padding controls.
  - **Sizing**: Adjustable sizing for avatars and containers.

## Character Grid Customization
- **Column Control**: Force the character grid to display 1 to 6 columns, overriding J.AI's responsive defaults.
- **Entity Styling**:
  - **Card Roundness**: Adjust border-radius for character cards.
  - **Grayscale Effect**: Make characters grayscale until hovered.
  - **Sheen Animation**: Add a premium "shiny" animation to names on hover.
  - **Hide Creator**: Option to hide the "By @user" text for a cleaner look.

## Rich Text Bio
- **HTML Snippets**: Quickly insert complex layouts like **Accordions** (collapsible sections) and **2-Column Text** blocks without writing code.
- **Live Preview**: WYSIWYG editing for bio content.
